# API for Managing


# MailBase PRO API is a server solution for creating and managing a database with customers emails.

Key fetures:
* Register and Authorization system
* Adding emails to the database
* sending letters to a database


## Usage Guide:

(work in progress)

## Installation guide

1. clone this repo
2. fill env.example with necessary data
3. run following command in your terminal

```
composer setup
```

4. For runnign test suit:

```
composer test
```
